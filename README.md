# cdc-foodborne-network-analysis
This project analyzes the hyperlink structure of the CDC Foodborne website using Python and NetworkX

# Libraries
- requests
- BeautifulSoup
- urllib.parse
- NetworkX

# Project Steps
1. Scrape webpages from the CDC Foodborne Outbreak section
2. Extract external hyperlinks
3. Calculate degree centrality, betweenness centrality, and PageRank
4. Visualize the network in Gephi
